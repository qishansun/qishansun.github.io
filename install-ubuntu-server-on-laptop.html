"Install Ubuntu Server on laptop"
Thu 17 September 2020 Server

1. wifi config:

apt install ubuntu-drivers-common
apt install networkmanager nmtui  
nmtui

then connect to wifi ap

2. Provent system suspend when lid closing:

https://www.dell.com/community/Linux-General/Stop-laptop-going-to-sleep-when-closing-the-lid-UBUNTU-Server/td-p/6086201

If you're not logged in as root, you'll need to add "sudo " to invoke the editor as in:

sudo nano /etc/systemd/logind.conf
In the file above, I changed only this line:

HandleLidSwitch=suspend

to this:

HandleLidSwitch=ignore
After saving the file, I ran:

sudo systemctl restart systemd-logind
This immediately resolved the problem.

3. Enable screen off after lid closing:

https://askubuntu.com/questions/1048063/ubuntu-server-how-do-disable-sleep-when-laptop-screen-is-closed-but-allow-scre

Since Ubuntu 18.04 doesn't have a /etc/rc.local file by default we need to create one.

sudo vi /etc/rc.local
press i to enter insert mode

Type the following 3 lines

1
2
3
#!/bin/bash    
setterm --blank 1 # here 1 means 1 minute 
exit 0

Next press escape, then type :wq to save and quit vi
Now change the permissions so the file can execute

sudo chown root /etc/rc.local
sudo chmod 755 /etc/rc.local

now restart the rc-local service. Type the following

systemctl restart rc-local
